# Prompt OS

A curated prompt library with expert-grade prompts across multiple categories — strategy, writing, productivity, learning, and more.

## Features

- Browse prompts by category or search across all
- One-click copy to clipboard
- Quality ratings for each prompt
- Variable placeholders highlighted for easy customization
- Dark, minimal UI with responsive layout

## Tech Stack

- Single-page static HTML/CSS/JS
- Hosted on [Vercel](https://vercel.com)
- No build step required

## Development

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
```

## Deployment

Deployed automatically via Vercel on push to `main`.
